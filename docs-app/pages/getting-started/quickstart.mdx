# Quick Start

Get SecureLens running in under 10 minutes with Docker Compose.

## Prerequisites

Before you begin, ensure you have:

- **Docker** 24.0+ with Docker Compose v2
- **8GB RAM** minimum (16GB recommended)
- **20GB disk space** for containers and data
- **Ports available**: 3000, 8080, 5432, 9092

## Step 1: Clone the Repository

```bash
git clone https://github.com/securelens/securelens.git
cd securelens
```

## Step 2: Configure Environment

Copy the example environment file:

```bash
cp .env.example .env
```

Edit `.env` with your settings:

```bash
# Required settings
SECURELENS_ADMIN_EMAIL=admin@yourcompany.com
SECURELENS_ADMIN_PASSWORD=YourSecurePassword123!

# Optional: External services
OPENAI_API_KEY=sk-...  # For AI Gate features
SMTP_HOST=smtp.yourcompany.com  # For notifications
```

## Step 3: Start Services

```bash
docker-compose up -d
```

This starts all required services:

| Service | Port | Description |
|---------|------|-------------|
| **SecureLens UI** | 3000 | Web interface |
| **SecureLens API** | 8080 | REST API |
| **PostgreSQL** | 5432 | Database |
| **Kafka** | 9092 | Event streaming |
| **Qdrant** | 6333 | Vector database |

## Step 4: Verify Installation

Check that all services are running:

```bash
docker-compose ps
```

Expected output:
```
NAME                    STATUS    PORTS
securelens-api          running   0.0.0.0:8080->8080/tcp
securelens-ui           running   0.0.0.0:3000->3000/tcp
securelens-postgres     running   0.0.0.0:5432->5432/tcp
securelens-kafka        running   0.0.0.0:9092->9092/tcp
securelens-qdrant       running   0.0.0.0:6333->6333/tcp
```

## Step 5: Access the UI

Open your browser and navigate to:

```
http://localhost:3000
```

Login with your configured credentials:
- **Email**: `admin@yourcompany.com`
- **Password**: `YourSecurePassword123!`

## Step 6: Connect Your First Data Source

1. Navigate to **Data Sources** in the sidebar
2. Click **Add Data Source**
3. Select your database type (PostgreSQL, MySQL, etc.)
4. Enter connection details
5. Click **Test Connection**
6. Click **Save & Scan**

SecureLens will automatically:
- Discover all tables and columns
- Classify sensitive data
- Generate a data map

## Troubleshooting

### Services won't start

Check Docker resources:
```bash
docker system info | grep -E "Memory|CPUs"
```

Ensure you have at least 8GB RAM allocated to Docker.

### Can't connect to UI

Verify the UI container is running:
```bash
docker-compose logs securelens-ui
```

### Database connection errors

Check PostgreSQL is healthy:
```bash
docker-compose exec postgres pg_isready
```
