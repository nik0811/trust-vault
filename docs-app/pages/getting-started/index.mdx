# Dashboard Overview

After logging in to SecureLens, you'll land on the main dashboard. This guide walks you through the key areas of the interface.

## First Login

When you first access SecureLens:

1. Go to [app.securelens.ai](https://app.securelens.ai)
2. Enter your email and password
3. Complete two-factor authentication (if enabled)

If you don't have an account yet, contact your organization's administrator or [request access](https://securelens.ai/request-demo).

## Dashboard Layout

The dashboard is organized into several key areas:

### Navigation Sidebar

The left sidebar provides access to all features:

| Section | What It Does |
|---------|--------------|
| **Dashboard** | Overview metrics and quick insights |
| **Data Sources** | Connect and manage your data connections |
| **Classification** | View and manage data classifications |
| **Data Map** | Visual inventory of your data |
| **Governance** | Policies, rules, and enforcement |
| **AI Gate** | LLM proxy settings and monitoring |
| **Privacy Center** | Data subject requests (DSAR) |
| **Audit Trail** | Access logs and activity history |
| **Reports** | Generate compliance reports |
| **Settings** | Users, API keys, and configuration |

### Key Metrics

The main dashboard displays real-time metrics:

- **Data Sources** — Number of connected sources and their health status
- **Classifications** — Total PII detected across all sources
- **Policies** — Active governance policies
- **Alerts** — Recent policy violations or issues requiring attention

### Quick Actions

Common tasks are accessible from the dashboard:

- **Add Data Source** — Connect a new database or storage system
- **Run Scan** — Trigger a scan on existing data sources
- **View Alerts** — See items requiring your attention
- **Generate Report** — Create a compliance report

## User Roles

Your access level depends on your assigned role:

| Role | What You Can Do |
|------|-----------------|
| **Admin** | Full access to all features, user management, settings |
| **Analyst** | View data, manage classifications, create reports |
| **Auditor** | Read-only access to audit logs and reports |
| **Developer** | API access, limited UI access |
