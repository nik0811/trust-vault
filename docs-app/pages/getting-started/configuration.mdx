# Configuration

Complete reference for SecureLens configuration options.

## Configuration Methods

SecureLens can be configured via:

1. **Environment variables** (recommended for containers)
2. **Configuration file** (`config.yaml`)
3. **Command-line flags**

Priority: CLI flags > Environment variables > Config file > Defaults

## Environment Variables

### Core Settings

| Variable | Description | Default |
|----------|-------------|---------|
| `SECURELENS_MODE` | Run mode: `gateway` or `worker` | `gateway` |
| `SECURELENS_PORT` | API server port | `8080` |
| `SECURELENS_LOG_LEVEL` | Log level: debug, info, warn, error | `info` |
| `SECURELENS_LOG_FORMAT` | Log format: json, text | `json` |

### Database

| Variable | Description | Default |
|----------|-------------|---------|
| `DATABASE_URL` | PostgreSQL connection string | Required |
| `DATABASE_MAX_CONNS` | Maximum connections | `25` |
| `DATABASE_IDLE_CONNS` | Idle connections | `5` |

### Kafka

| Variable | Description | Default |
|----------|-------------|---------|
| `KAFKA_BROKERS` | Comma-separated broker list | Required |
| `KAFKA_CONSUMER_GROUP` | Consumer group ID | `securelens` |
| `KAFKA_TOPIC_PREFIX` | Topic name prefix | `securelens.` |

### Authentication

| Variable | Description | Default |
|----------|-------------|---------|
| `JWT_SECRET` | JWT signing secret (min 32 chars) | Required |
| `JWT_EXPIRY` | Token expiry duration | `24h` |
| `API_KEY_ENABLED` | Enable API key auth | `true` |

### Classification

| Variable | Description | Default |
|----------|-------------|---------|
| `GLINER_MODEL_PATH` | Path to GLiNER model | `/models/gliner` |
| `CLASSIFICATION_BATCH_SIZE` | Records per batch | `1000` |
| `CLASSIFICATION_WORKERS` | Parallel workers | `4` |

## Configuration File

Create `config.yaml` in the working directory:

```yaml
server:
  port: 8080
  mode: gateway
  
logging:
  level: info
  format: json
  
database:
  url: postgres://user:pass@localhost:5432/securelens
  maxConns: 25
  idleConns: 5
  
kafka:
  brokers:
    - kafka:9092
  consumerGroup: securelens
  topicPrefix: securelens.
  
auth:
  jwtSecret: your-secret-key
  jwtExpiry: 24h
  apiKeyEnabled: true
  
classification:
  modelPath: /models/gliner
  batchSize: 1000
  workers: 4
  
aiGate:
  enabled: true
  defaultProvider: openai
  providers:
    openai:
      apiKey: ${OPENAI_API_KEY}
      model: gpt-4
    azure:
      endpoint: ${AZURE_OPENAI_ENDPOINT}
      apiKey: ${AZURE_OPENAI_KEY}
```

## Feature Flags

Enable or disable features:

```yaml
features:
  classification: true
  aiGate: true
  dataQuality: true
  rotDetection: true
  compliance: true
  notifications: true
```

## Multi-Tenant Configuration

```yaml
multiTenant:
  enabled: true
  isolation: strict  # strict, shared
  defaultTenant: default
```
