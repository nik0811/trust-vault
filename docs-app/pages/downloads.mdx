### Verify Download

Always verify the checksum after downloading:

**Linux / macOS:**
```bash
# Download
curl -L -o securelens-agent https://api.securelens.ai/api/v1/downloads/agent/linux-amd64

# Verify checksum
echo "854265223d5dc68bac17da924f2ccde0029f92cb861ddd662d7fd0de137aea58  securelens-agent" | shasum -a 256 -c
```

**Windows (PowerShell):**
```powershell
# Download
Invoke-WebRequest -Uri "https://api.securelens.ai/api/v1/downloads/agent/windows-amd64" -OutFile "securelens-agent.exe"

# Verify checksum
(Get-FileHash securelens-agent.exe -Algorithm SHA256).Hash -eq "aaa4b56855dfe86a30cb9e130f26b32983c84ea0e2e00d074fcc182fb3e7235d"
```

### Quick Install

**Linux (x86_64):**
```bash
curl -L -o securelens-agent https://api.securelens.ai/api/v1/downloads/agent/linux-amd64
chmod +x securelens-agent
sudo mv securelens-agent /usr/local/bin/
securelens-agent init --api-key YOUR_API_KEY
```

**Linux (ARM64):**
