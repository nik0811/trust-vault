# Dashboard

The SecureLens dashboard provides a real-time overview of your data governance posture.

![Dashboard](/screenshots/dashboard.png)

## What You'll See

When you log in, the dashboard displays key metrics at a glance:

- **Data Sources** — Connected sources and their health status
- **Classifications** — PII detected across all sources
- **Policy Violations** — Recent governance violations
- **Data Quality** — Overall quality scores

## Key Metrics

### Data Discovery

| Metric | What It Means |
|--------|---------------|
| **Total Records** | Records scanned across all sources |
| **Classified Records** | Records with sensitivity labels |
| **PII Detected** | Count of PII instances found |
| **Coverage** | Percentage of data classified |

### Governance

| Metric | What It Means |
|--------|---------------|
| **Active Policies** | Policies currently enforced |
| **Violations (24h)** | Policy violations in last 24 hours |
| **Blocked Requests** | AI Gate blocked requests |
| **Compliance Score** | Overall compliance percentage |

### Data Quality

| Metric | What It Means |
|--------|---------------|
| **Quality Score** | Average data quality (0-100) |
| **ROT Detected** | Redundant/obsolete/trivial data |
| **Issues Found** | Data quality issues |
| **Remediation Rate** | Issues fixed vs. found |

## Dashboard Widgets

### Classification Breakdown

A pie chart showing the distribution of sensitive data types:

- Personal Information (PII)
- Financial Data (PCI)
- Health Information (PHI)
- Confidential Business Data
- Public Data

### Trend Charts

Line charts showing trends over time:

- Classifications per day
- Policy violations per day
- AI Gate requests per day
- Data quality score trend

### Recent Activity

A live feed of recent events:

- New classifications
- Policy violations
- Data source scans
- User actions

### Alerts

Active alerts requiring your attention:

- Critical policy violations
- Data source connection issues
- Classification anomalies
- System health warnings

## Customizing Your Dashboard

### Add or Remove Widgets

1. Click **Customize** in the top right
2. Select widgets from the catalog
3. Drag to arrange the layout
4. Click **Save**

### Change Date Range

Adjust the time range for metrics:

- Last 24 hours
- Last 7 days
- Last 30 days
- Custom range

### Export Dashboard Data

Export dashboard data for reporting:

- **PDF** — Full dashboard snapshot
- **CSV** — Raw metrics data

## Common Use Cases

### Morning Check-In

Start your day by reviewing:
1. Any new alerts or violations
2. Overnight scan results
3. Compliance score changes

### Preparing for Audits

Use the dashboard to:
1. Check compliance scores by framework
2. Review recent policy violations
3. Export metrics for audit documentation

### Monitoring AI Usage

Track AI Gate activity:
1. View requests and token usage
2. Check for PII detection in prompts
3. Monitor costs by team or project
