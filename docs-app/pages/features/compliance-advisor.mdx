# Compliance Advisor

AI-powered compliance guidance and gap analysis.

## Overview

Compliance Advisor provides:

- **Gap Analysis**: Identify compliance gaps
- **Recommendations**: Actionable remediation steps
- **Progress Tracking**: Monitor compliance journey
- **Reporting**: Generate audit-ready reports

## Custom Frameworks

### Create Custom Framework

1. Navigate to **Compliance Advisor** → **Frameworks**
2. Click **Create Framework**
3. Define controls:

```yaml
name: "Internal Data Policy"
version: "2.0"
controls:
  - id: IDP-001
    name: "Data Classification"
    description: "All data must be classified within 24 hours"
    category: "Data Governance"
    evidence_types:
      - classification_report
      - sla_metrics
      
  - id: IDP-002
    name: "Access Review"
    description: "Access must be reviewed quarterly"
    category: "Access Control"
    evidence_types:
      - access_review_report
```
