# Reports

Generate compliance, operational, and executive reports.

## Report Types

### Compliance Reports

| Report | Description | Audience |
|--------|-------------|----------|
| **GDPR Summary** | EU compliance status | DPO, Auditors |
| **CCPA Summary** | California compliance | Privacy Team |
| **HIPAA Summary** | Healthcare compliance | Compliance |
| **PCI-DSS Summary** | Payment card compliance | Security |
| **SOC 2 Evidence** | Audit evidence package | Auditors |

### Operational Reports

| Report | Description | Audience |
|--------|-------------|----------|
| **Classification Summary** | PII detection results | Data Team |
| **Scan Report** | Data source scan results | Operations |
| **Quality Report** | Data quality metrics | Data Team |
| **ROT Analysis** | Redundant data findings | Storage Team |

### Executive Reports

| Report | Description | Audience |
|--------|-------------|----------|
| **Executive Summary** | High-level overview | Leadership |
| **Risk Dashboard** | Risk posture summary | CISO |
| **Trend Analysis** | Month-over-month trends | Leadership |

## Generating a Report

### On-Demand Reports

1. Go to **Reports**
2. Select the report type
3. Configure parameters:
   - Time period
   - Data sources to include
   - Output format
4. Click **Generate**
5. Download when ready

### Scheduled Reports

Set up automatic report generation:

1. Go to **Reports** → **Scheduled**
2. Click **Create Schedule**
3. Select report type
4. Set the schedule (daily, weekly, monthly)
5. Configure delivery (email, Slack)
6. Click **Save**

## Report Delivery

Reports can be delivered via:

- **Email** — Send to recipients
- **Slack** — Post to a channel
- **Download** — Manual download from UI

## Customizing Reports

### Branding

Add your company branding:

1. Go to **Settings** → **Branding**
2. Upload your logo
3. Set primary colors
4. Configure footer text

### Custom Reports

Create reports tailored to your needs:

1. Go to **Reports** → **Custom**
2. Click **Create Report**
3. Add sections (metrics, charts, tables)
4. Configure data sources
5. Save and generate

## Common Use Cases

### Preparing for an Audit

1. Generate the relevant compliance report (GDPR, HIPAA, etc.)
2. Export audit trail for the audit period
3. Include evidence of policy enforcement

### Monthly Executive Update

1. Generate Executive Summary report
2. Include trend analysis
3. Highlight key achievements and risks

### Data Team Review

1. Generate Classification Summary
2. Review new PII discoveries
3. Check data quality scores
